## ディスクを確認

```console
$ df

$ cat /etc/vfstab
```


# PROMモニタへ移行

```console
# init 0
```

でシステム停止する.
(Sunキーボードは「Stopキー + a」でもPROMモニタへ移行できる.)
そうすると, okプロンプトになる.

```
ok
```


## CDブート

CDドライブにSolarisのインストールメディアを挿入する.
okプロンプトから,

```
ok boot cdrom -s
```

により, シングルユーザでCDブートする.


## フォーマット

起動後, 以下のようにディスクを`format`する.

```console
# format
Specify disk (enter its number): X
format> analyze
analyze> purge
```
