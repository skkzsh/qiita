## 設定

`~/.conkyrc`に以下を追記する.

```~/.conkyrc
[Global]
${texeci 3600 curl ifconfig.me}
${texeci 3600 curl ifconfig.me/host}
```


## 説明

`$texeci`により, 指定した時間間隔でコマンドを実行して, その出力を**Conky**に表示できる.
`texeci`の後の数字は, 実行するコマンドの時間間隔 (秒) で, 適宜編集する.

なお,

- **グローバルIPアドレス**は`curl ifconfig.me`,
- **ホスト名**は`curl ifconfig.me/host`,

により, 取得できる.
