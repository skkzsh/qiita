MATLABのコードをコマンドラインからは,
以下のようにして指定して実行できる.

```sh
echo 'magic(3)' | matlab -nodisplay
```

MATLABのコードをシェルスクリプト内で指定するときにも便利.

```sh
for lambda in `seq 0 0.2 1` ; do
    echo \
    "my_function($lambda);" \
    | matlab -nodisplay
done
```
