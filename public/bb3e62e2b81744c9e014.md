## 設定

上の中から, 例えば, `terraform`, `pip`のオプション補完を追加する場合は,
以下のように`$fpath`を追加する.
(ただし, `$fpath`は`compinit`より前に書く. )

```bash:~/.zshrc
# fpathの追加
for cmd in terraform pip ; do
    fpath=(/path/to/ohmyzsh/plugins/$cmd $fpath)
done
unset cmd

# 補完を有効化
autoload -Uz compinit
compinit
```

`~/.zshrc`の再読み込み後, これらのコマンドでオプション補完が有効になる.
