このハイライトは, syntax file

```vim
$VIMRUNTIME/syntax/*.vim
```

で定義されている.
よって, filetypeごとにハイライトされる文字が異なる.

ざっと見たところ, 以下の文字がよくハイライトされる.

> TODO FIXME XXX NOTE

詳しくは, 各filetypeのsyntax file

```vim
$VIMRUNTIME/syntax/*.vim
```

を見る.


## 参考

なお, [`$VIMRUNTIME`](https://vim-jp.org/vimdoc-ja/starting.html#$VIMRUNTIME)の値はVimで,

```vim:Vim
:echo $VIMRUNTIME
```

とすれば, 確認できる.

現在のバッファの[filetype](https://vim-jp.org/vimdoc-ja/filetype.html)は

```vim:Vim
:set filetype
```

で確認できる.

例えば,

```vim
filetype=python
```

のsyntaxは,

```vim
$VIMRUNTIME/syntax/python.vim
```

で定義されている.
